💡 Issue pr 제목 형식

- [FEAT] 기능구현 (eg. 회원가입 구현, UI 구현)
- [HOTFIX] 당장 0순위로 고쳐야 하는 상황
- [FIX] 코드 수정
- [DEBUG] 버그 발견 및 수정
- [STYLE] 폰트 변경
- [ADD] Asset 추가
- [CHORE] 빌드 테스트, 패키지 매니저 설정 등 코드 수정이 없을 때
